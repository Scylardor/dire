Detailed Description

template<typename... Ts>
struct DIRE_NS::Metadata< Ts >

The holder class for Dire property metadatas. Aims to provide a functionality similar to Unity's attributes or Unreal's UMETA (or meta keywords), by tagging each property. In its current iteration, Metadata is supposed to be used only with constexpr types and values, which is why it just acts as a holder of a variadic type list. It is only useful in the context of serialization, and as of now, is only useful for JSON serializing. Its purposes is not to store in-game data.

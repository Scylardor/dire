Detailed Description

The base class for manipulating reflection data. Inherit from it with the dire_reflectable(...) macro. Allows you to get, set and clone properties, get member function information, and dynamically test for class relationship. Keep in mind this class is virtual and will add a virtual table to any type inheriting from it.

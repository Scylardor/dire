Detailed Description

template<typename T, bool UseDefaultCtorForInstantiate>
class DIRE_NS::TypedTypeInfo< T, UseDefaultCtorForInstantiate >

The metadata type declared by DIRE_TYPE_INFO and DIRE_REFLECTABLE_INFO. Never use directly, always use one these two macros. The boolean parameter lets you decide if this class should use its default constructor as an instantiator function. DIRE's default is true.

Template Parameters
TThe type
UseDefaultCtorForInstantiateTrue if this class should register its default constructor as instantiator (true by default)

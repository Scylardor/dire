Detailed Description

The database for every type info in the program. It is supposed to be used as a singleton (except for testing purposes), thus you should only have one in your whole program. It allows you to fetch any type info or instantiate any Reflectable type, given you provide the matching Reflectable ID. It also supports importing and exporting the type info database, which is useful for persistent identification of types across multiple runs of a program.
